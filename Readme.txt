Локальний запуск програми.
1. локально збережіть папку "PortaOne quiz"; 
2. відкрийте папку "PortaOne quiz" за допомогою редактора коду (VS Code, як варіант);
3. в терміналі редактора коду напишіть команду "npx http-server"

Сторінка буде доступна за адресами:
  http://192.168.1.3:8080
  http://127.0.0.1:8080
Hit CTRL-C to stop the server
